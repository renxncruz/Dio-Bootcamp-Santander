< h1 > ao < h6 >

Essas tags são super importantes para estrutura e para semântica de qualquer pagina web.
Basicamente, elas definem os níveis de importância dos títulos no seu contudo, sendo a h1 para o título principal, pois é a maior, e h6 sendo o nível de menos importância, pois é o menor.

Exemplos abaixo:

h1

h2

h3

h4

h5
h6

< p >

Essa é uma tag para criar parágrafos de texto.

< mark >

Essa tag serve para destacar uma parte do texto como ser tivesse passado um marca-texto amarelo.

Exemplo: Dessa forma!

< small >

É utilizada para texto que é menos importante ou que é um comentario secundário, tipo um aviso de direitos autorais ou uma nota de rodapé.

Exemplo: Lorem ipsum dolor sit, amet consectetur...

< i >

Deixa o texto em itálico. Geralmente usado para termos técnicos, nomes científicos ou quando voce quer dar uma ênfase leve.

Exemplo: Aqui temos um texto em itálico.

< u >

Essa tag é utilizada para sublinhar o texto. Temos que tomar cuidado para não confundir com links, porque links também são sublinhados por padrão.

Exemplo: Esse é um texto utilizando a tag u!

< strong >

A tag < strong > é utilizada para dar ênfase e importância semantica a um trecho de texto, deixando ele em negrito.
Importante: As tags <strong> e <b>, fazem o texto parecer em negrito por padrão, mas a diferença fundamental está no significado semântico. Enquanto a tag <strong> indica que o conteúdo dentro dela tem forte importância, a tag <b> serve para chamar a atenção para um texto sem necessariamente indicar que ele tem uma importância maior.

Exemplo: Nossa reunião amanhã será às 10h da manhã !

< ol >

Essa tag é usada para criar listas e significa a palavra Ordered List (Lista Ordenada).
É utilizada como "uma sequência" onde a ordem é importante, e cada item da lista é marcado com um número, uma letra ou um algarismo romano.
Usamos essa tag < ol > para envolver todos os itens da lista nao ordenada.

Exemplo:

<ol>
<li>Primeiro passo: Abrir o navegador</li>
<li>Segundo passo: Digitar o endereço do site</li>
<li>Terceiro passo: Digitar enter</li>
</ol>
  1. Primeiro passo: Abrir o navegador
  2. Segundo passo: Digitar o endereço do site
  3. Terceiro passo: Digitar enter

< ul >

Essa tag é utilizada para criar listas e significa a palavra Unordered List (Lista Não Ordenada).
É utilizada como uma "lista de itens" onde a ordem não importa e cada item dessa lista é marcado com um ponto, um quadrado ou um círculo.
Usamos essa tag < ul > para envolver todos os itens da lista não ordenada.

Exemplo:

<ul>
<li>Primeiro passo: Abrir o navegador</li>
<li>Segundo passo: Digitar o endereço do site</li>
<li>Terceiro passo: Digitar enter</li>
</ul>
  • Primeiro passo: Abrir o navegador
  • Segundo passo: Digitar o endereço do site
  • Terceiro passo: Digitar enter

< li >

A tag <li> significa a palavra List Item (Item da lista).
Sempre será utilizada dentro de uma tag <ul> ou <ol>, pois ela não funciona sozinha e cada <li> é um elemento separado na sua lista.

< a >

A tag <a> que significa "ancora" (anchor em inglês), é uma das mais importantes do HTML.
Imagine ela como uma ponte onde conectar sua página a outras páginas da web, partes específicas da mesma página, arquivos de downloads, entre outros.
Para ela funcionar, é necessário um atributo chamado href. O href é onde você especifica o destino do seu link. Ou seja, para onde o usuário será levado quando clicar no link.

Exemplo de estrutura básica: <a href="destino_do_link">Texto que o usuário vai clicar</a>

< hr >

A tag <hr> no HTML é bem simples, mas muito útil!
Ela é uma tag vazia (o que significa que não precisa de uma tag de fechamento) e serve para criar uma linha horizontal na sua página, que tem como principal função, separar seções temáticas de conteúdo.

< sub >

Essa tag é utilizada para exibir subscrito, que é um texto ligeiramente menor e posicionado um pouco abaixo da linha de base do texto normal., e normalmente é utilizado para usos bem específicos e técnicos, principalmente em fórmulas quimicas, notações matemáticas e alguns tipos de notações científicas.

Exemplo: H2O

< sup >

Essa tag é utilizada para exibir sobrescrito, assim como a tag sub, ela exibe um texto ligeiramente menor, porém é posicionado acima da linha de base do texto normal.

Exemplo: 1a

< blockquote >

Essa tag é usada para citações longas que precisam ser destacadas do restante do texto. Ela geralmente cria um bloco de texto recuado.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Architecto nulla cupiditate dolor, fugiat quae odit rerum, laboriosam minima eius maxime nesciunt quo, esse debitis beatae consequuntur recusandae neque. Vitae, aspernatur.

< font >

A tag <font> hoje em dia não é recomendada para uso em desenvolvimento web moderno e está obsoleta.
Ela usada para estilizar o texto diretamente no HTML e poderiamos controlar coisas como, color (cor do texto), size (tamanho da fonte) e face (família da fonte).
Hoje em dia a forma correta e moderna de estilizar texto e qualquer outro elemento HTML é usando CSS.

< del >

A tag <del> é usada para representar um texto que foi excluído (deletado) de um documento. Ela não remove o texto da visualização, mas o exibe com um risco no meio (tachado), indicando que aquele conteúdo não é mais valido.
O principal objetivo da tag é fornecer semântica (significado) ao conteúdo, indicando correçoes ou atualizações.

Exemplo: O preço original era de R$100,00 agora é de R$75,00!

< abbr >

Essa tag é usada para indicar que um texto é uma abreviação ou um acrônimo.
Ela nao altera a aparência visual do texto por padrão, mas fornece informações adicionais importantes para navegadores, ferramentas de busca e, principalmente, para acessibilidade.

Exemplo:
O Brasil é um país membro da ONU.

Uma das principais utilidades dessa tag é melhorar a acessibilidade, para que leitores de tela possam ler a abreviação corretamente. Isso é crucial para entender o contexto. Por exemplo, em vez de ler "o-n-u", ele poderá ler "Organização das Nações Unidas". Além disso, melhora a usabilidade para todos usuários, pois quando ele passa o mouse sobre a abreviação, o texto completo do title aparece como uma pequena caixa de texto flutuante, permitindo que o usuário veja o significado sem precisar pesquisar ou adivinhar.
